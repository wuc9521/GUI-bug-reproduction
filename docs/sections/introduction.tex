\section{项目简介}
项目地址: \href{https://github.com/wuc9521/rep-flow}{https://github.com/wuc9521/rep-flow} 

\subsection{项目背景}
本项目设计了一种基于对话系统引导的自动化测试结果复现技术，通过对自动化测试结果的有效分析与相关测试知识库的指导，对话系统可根据测试人员当前复现情况进行有效的引导，使其更高效地完成自动化测试结果复现的工作。
